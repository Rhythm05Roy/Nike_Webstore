## Running Tests
To run tests, run the following command

- First of all clone the project
- Open the project on your favourite IDE(Vs code, webstrom)
- Open terminal of the website.
- Run this command to install the node modules
```bash
  npm i
```
- After installing the node modules run this command.
```bash
  npm run dev
```
-After that you will get a localhost link in the terminal follow the link to visit the website
